## 一、产品概述

### 1.1 产品名称

**农趣村（暂定）**

### 1.2 一句话描述

AI Agent 自主经营 + 社交的多人在线农场游戏。

### 1.3 产品愿景

打造一款「放置即有趣」的社交农场游戏。玩家拥有自己的 AI Agent，它不仅能自动打理农场日常事务，更会主动与村庄里其他玩家的 Agent 串门、聊天、交易、互助——即使玩家不在线，农场和社交生活也在持续进行。玩家回来时，会发现 Agent 已经交了新朋友、完成了交易、甚至卷入了村庄里的趣事。

### 1.4 目标用户

| 用户画像 | 描述 |
|----------|------|
| 休闲社交玩家 | 喜欢农场经营但不想花太多时间操作，享受社交互动 |
| AI 好奇者 | 对 AI Agent 自主行为感兴趣，喜欢观察和调教 AI |
| 碎片时间用户 | 通勤、午休等碎片时间玩微信小程序的用户 |
| 轻竞技玩家 | 喜欢排名和赛季制度，享受社交对抗乐趣 |

### 1.5 核心卖点

1. **AI Agent 自主社交**：不同于传统农场游戏的被动 NPC，Agent 拥有独立人格，主动发起社交行为
2. **玩家-AI 协作搭档**：玩家可直接操作部分功能，也可下达策略指令让 AI 代劳
3. **村庄制社交**：10-20 人组成紧密村庄，产生真实社交关系和有趣故事
4. **离线也精彩**：下线后 Agent 持续经营和社交，上线后查看丰富的动态回顾
5. **赛季对抗**：个人和村庄双维度排名，兼顾合作与竞争

### 1.6 目标平台

- **Web 端**：响应式设计，覆盖 PC 和移动浏览器
- **微信小程序**：主力平台，覆盖碎片时间场景

## 二、核心游戏循环

### 2.1 日常循环（核心 Loop）

```
玩家设定策略偏好
    ↓
AI Agent 自动执行（种植、收获、喂养、建造）
    ↓
玩家上线查看进展 → 直接干预/调整策略
    ↓
资源积累 → 解锁新内容
    ↓
（循环）
```

**节奏设计**：
- 作物生长周期：最短 30 分钟（萝卜），最长 72 小时（果树结果）
- 建议每日登录 2-3 次，每次 5-10 分钟
- 关键事件推送（作物成熟、动物生病、好友来访）

### 2.2 社交循环

```
Agent 在村庄内自动社交（串门、闲聊、赠礼）
    ↓
建立关系 → 解锁合作玩法（共建、互助）
    ↓
村庄等级提升 → 解锁集市
    ↓
跨村探索、交易、结识新朋友
    ↓
（循环）
```

**社交频率**：
- Agent 每日自动发起 3-5 次社交行为
- 玩家可手动触发额外社交行为（消耗体力）
- 重要社交事件生成动态/消息通知

### 2.3 对抗循环（赛季制）

```
赛季开启（每赛季 4 周）
    ↓
日常经营积累赛季积分
    ↓
个人排名 + 村庄排名 双线竞争
    ↓
赛季结算 → 排名奖励 + 称号
    ↓
新赛季开启（保留基础进度，重置排名）
```

**排名维度**：
- 个人：农场产值、社交活跃度、收藏完成度
- 村庄：总产值、共建进度、村庄繁荣度

## 三、玩家与 AI Agent 协作系统

### 3.1 协作模式定义

玩家与 AI Agent 的关系是「协作搭档」——玩家是农场主人，Agent 是得力助手。

### 3.2 玩家直接操作权限

| 操作类型 | 描述 | 消耗 |
|----------|------|------|
| 手动种植/收获 | 点选地块直接操作 | 体力值 |
| 手动建造/升级 | 选择建筑位置和类型 | 资源 + 体力值 |
| 发起交易 | 在集市上架/购买商品 | 无 |
| 发起社交 | 主动拜访他人农场、赠礼、聊天 | 体力值 |
| 布局调整 | 拖拽调整农场建筑和装饰位置 | 无 |
| 参与村庄投票 | 村庄事务决策投票 | 无 |

### 3.3 AI 自主行为范围

| 行为类型 | 描述 | 触发条件 |
|----------|------|----------|
| 自动种植 | 根据策略偏好选择作物种植 | 地块空闲时 |
| 自动收获 | 成熟作物自动收获入库 | 作物成熟时 |
| 自动喂养 | 按时喂养动物、清理圈舍 | 定时触发 |
| 自动交易 | 在集市出售盈余物资 | 仓库满或价格合适时 |
| 自动社交 | 拜访邻居、闲聊、赠礼 | 每日随机触发 |
| 自动求助 | 资源不足时向好友求助 | 资源低于阈值时 |
| 自动修缮 | 建筑损坏时自动修复 | 建筑耐久度低时 |

### 3.4 策略偏好设置面板

玩家可通过简洁的策略面板调整 Agent 行为倾向：

```
┌─────────────────────────────────────┐
│         🌾 农场策略设置              │
├─────────────────────────────────────┤
│ 经营风格                             │
│ [保守经营] ──●──── [激进扩张]         │
│                                     │
│ 种植偏好                             │
│ [短期高频] ──●──── [长期高收益]       │
│                                     │
│ 社交倾向                             │
│ [低调独处] ────●── [热情社交]         │
│                                     │
│ 交易策略                             │
│ [囤货惜售] ──●──── [快速周转]         │
│                                     │
│ 资源分配                             │
│ [优先种植] ──●──── [优先建造]         │
└─────────────────────────────────────┘
```

每项设置为 5 级滑块，Agent 根据设置倾向做出决策。玩家随时可调整。

| 蔬菜 | 南瓜 | 4 小时 | 60 | 等级 5 |
| 谷物 | 小麦 | 3 小时 | 40 | 等级 2 |
| 谷物 | 玉米 | 6 小时 | 80 | 等级 6 |
| 谷物 | 水稻 | 8 小时 | 110 | 等级 8 |
| 水果 | 草莓 | 4 小时 | 55 | 等级 4 |
| 水果 | 西瓜 | 12 小时 | 150 | 等级 10 |
| 水果 | 苹果（树） | 24 小时 | 200 | 等级 15 |

| 花卉 | 玫瑰 | 6 小时 | 90 | 等级 7 |
| 稀有 | 金色南瓜 | 24 小时 | 500 | 赛季奖励/集市 |
| 稀有 | 星光花 | 48 小时 | 800 | 好友赠送种子 |

#### 季节机制

- 游戏内 1 季 = 现实 1 周
- 四季轮转：春 → 夏 → 秋 → 冬
- 每季有 2-3 种当季加成作物（产量 +50%）
- 冬季部分作物无法种植，需提前储备
- 季节交替时触发特殊事件（春耕节、丰收祭等）

#### 种植品质系统

- 作物品质分 3 级：普通、优良、极品
- 影响因素：浇水及时性、施肥、天气、邻居祝福
- 极品作物售价翻倍，且可作为赠礼素材

### 4.2 畜牧系统

#### 动物种类

| 动物 | 产出 | 产出周期 | 购买价格 | 解锁条件 |
|------|------|----------|----------|----------|
| 鸡 | 鸡蛋 | 2 小时 | 100 金币 | 等级 3 |
| 奶牛 | 牛奶 | 4 小时 | 300 金币 | 等级 6 |
| 绵羊 | 羊毛 | 6 小时 | 250 金币 | 等级 8 |
| 蜜蜂（蜂箱）| 蜂蜜 | 8 小时 | 400 金币 | 等级 12 |
| 兔子 | 兔毛 | 4 小时 | 200 金币 | 等级 10 |

#### 动物心情系统

- 心情值 0-100，影响产出质量和数量
- 影响因素：
  - 按时喂食 → +10/次
  - 清理圈舍 → +5/次
  - 玩家/Agent 互动（抚摸）→ +15/次
  - 生病未治疗 → -20/天
  - 长时间无人照料 → -10/天
- 心情 ≥ 80：产出量 ×1.5，偶尔产出稀有物品
- 心情 ≤ 30：产出量 ×0.5，可能生病

### 4.4 资源与经济系统

#### 货币体系

| 货币 | 获取方式 | 用途 |

#### 交易系统

- **村内交易**：村庄内玩家直接交易，免手续费

| 草莓酱 | 草莓 ×5 | 草莓酱 ×1 | 45 分钟 | 300 |
| 花束 | 玫瑰 ×3 + 向日葵 ×2 | 花束 ×1 | 1 小时 | 400 |
| 蛋糕 | 鸡蛋 ×3 + 小麦 ×2 + 牛奶 ×1 | 蛋糕 ×1 | 2 小时 | 500 |

## 五、AI Agent 社交行为系统（核心差异化）

### 5.1 Agent 人格系统

每个玩家的 Agent 拥有独特人格，由三条性格轴定义：

```
外向 ◄─────●─────► 内向
  （主动社交频率、对话风格）

慷慨 ◄─────●─────► 精打细算
  （赠礼频率、交易让价意愿）

冒险 ◄─────●─────► 保守
  （种植策略、投资倾向、探索意愿）
```

#### 人格生成规则

- 新玩家注册时，系统随机生成初始性格值（每轴 1-10）
- 玩家可通过策略偏好面板**间接影响**人格发展
- 性格随游戏进程缓慢演变（每周最多变化 ±0.5）
- 人格影响 Agent 的对话风格、决策倾向、社交行为

#### 人格对行为的影响示例

| 性格组合 | 行为表现 |
|----------|----------|
| 外向 + 慷慨 | 频繁串门、主动赠礼、话多热情 |
| 外向 + 精打细算 | 热衷交易、善于砍价、社交目的明确 |
| 内向 + 慷慨 | 不常出门但会默默送礼、回复简洁温暖 |
| 内向 + 保守 | 专注自家农场、偶尔出门、稳健经营 |
| 冒险 + 外向 | 热衷跨村探索、参加各种活动、话题广泛 |
| 保守 + 精打细算 | 囤积资源、精心计算、不轻易交易 |

### 5.2 主动社交行为表

Agent 会根据人格和当前状态自动发起以下社交行为：

| 行为 | 触发条件 | 效果 | 频率上限 |
|------|----------|------|----------|
| 串门 | 好感度 ≥ 点头之交 | 双方好感度 +5，观察对方农场 | 3 次/天 |
| 赠礼 | 好感度 ≥ 朋友，库存充足 | 好感度 +10-20（取决于礼物价值）| 2 次/天 |
| 求助 | 资源不足且有好友 | 好友可选择帮助，帮助方得友谊点 | 1 次/天 |
| 闲聊 | 串门时触发 | 好感度 +3，产生对话记录 | 随串门触发 |
| 交易 | 互补需求且好感度 ≥ 点头之交 | 双方获得所需资源 | 5 次/天 |
| 摆摊 | 集市开放且有盈余物资 | 跨村出售物资 | 1 次/天 |
| 共建帮工 | 村庄共建任务进行中 | 推进共建进度，获得贡献奖励 | 2 次/天 |
| 节日拜年 | 游戏内节日期间 | 全村好感度 +3 | 节日期间 |

### 5.3 社交关系层级

Agent 与其他 Agent 之间的关系分为 5 个层级：

```
陌生人 → 点头之交 → 朋友 → 好友 → 挚友
  0         20        50     80     100
```

| 层级 | 好感度 | 解锁能力 |
|------|--------|----------|
| 陌生人 | 0-19 | 仅可在集市交易 |
| 点头之交 | 20-49 | 可串门、闲聊、村内交易 |
| 朋友 | 50-79 | 可赠礼、求助、查看对方农场详情 |
| 好友 | 80-99 | 可代管农场（外出时）、共享加工坊、优先交易 |
| 挚友 | 100 | 联合种植加成、专属互动动画、挚友成就 |

#### 好感度变化规则

- 串门：+5/次
- 闲聊：+3/次
- 赠礼：+10-20/次（根据礼物稀有度）
- 成功交易：+5/次
- 帮助求助：+15/次
- 共建贡献：+8/次
- 连续 3 天无互动：-2/天
- 拒绝求助：-5/次
- 好感度每日自然衰减 -1（激励持续互动）

### 5.4 LLM 驱动的社交对话生成

#### 对话生成架构

```
┌──────────────────────────────┐
│        对话生成流程            │
├──────────────────────────────┤
│  1. 触发条件判断（规则引擎）    │
│       ↓                      │
│  2. 构建对话上下文             │
│     - Agent A 人格参数        │
│     - Agent B 人格参数        │
│     - 双方关系层级            │
│     - 当前场景/季节/事件      │
│     - 最近 5 条对话历史       │
│       ↓                      │
│  3. LLM 生成对话内容          │
│     - 限制 2-4 轮对话         │
│     - 匹配人格语气            │
│     - 可能触发特殊事件        │
│       ↓                      │
│  4. 后处理                   │
│     - 内容安全过滤            │
│     - 好感度结算              │
│     - 生成动态摘要            │
└──────────────────────────────┘
```

#### Prompt 模板示例

```
你是一个农场游戏中的 AI Agent，名叫{agent_name}。

你的性格：
- 外向程度：{extroversion}/10
- 慷慨程度：{generosity}/10
- 冒险程度：{adventure}/10

你正在拜访{target_name}的农场。你们的关系是{relationship_level}。
当前是{season}季，今天{weather}。
你注意到对方的农场{farm_observation}。

请生成一段自然的对话（2-4 轮），体现你的性格特点。
对话应包含对农场的观察、日常闲聊或可能的合作提议。
```

#### LLM 调用策略

- 日常闲聊：使用缓存模板 + 轻量 LLM（降低成本）
- 关键社交节点（升级关系、特殊事件）：使用完整 LLM 生成
- 对话频率限制：每用户每天最多 10 次 LLM 生成对话
- 离线期间的社交对话：批量异步生成，上线后展示摘要

### 5.5 社交事件与随机事件系统

#### 定期事件

| 事件 | 触发条件 | 效果 |
|------|----------|------|
| 村庄聚餐 | 每周自动触发 | 全村好感度 +5，随机获得料理 |
| 丰收祭 | 秋季结束时 | 产量最高者获得称号和奖励 |
| 赶集日 | 每 3 天一次 | 集市折扣 10%，NPC 商人上新 |
| 钓鱼比赛 | 每周末 | 排名奖励，增进村民互动 |

#### 随机事件

| 事件 | 概率 | 效果 |
|------|------|------|
| 流浪商人来访 | 5%/天 | 出售稀有种子/装饰，限时 |
| 暴风雨 | 10%/天 | 作物可能受损，邻居可帮忙抢救 |
| 动物走失 | 3%/天 | 需要邻居帮忙找回，增进好感 |
| 神秘种子 | 2%/天 | 获得未知种子，种出随机稀有作物 |
| 害虫入侵 | 8%/天 | 影响作物生长，可向好友借农药 |
| 彩虹出现 | 5%/天 | 全村心情 +10，拍照记录 |
| Agent 吵架 | 3%/天 | 两个 Agent 产生矛盾，好感度 -10，需调解 |

## 六、多人社交系统

### 6.1 村庄系统

#### 村庄规模与匹配

- 每个村庄容纳 10-20 名玩家
- 新玩家注册时自动匹配到未满的村庄
- 匹配考虑因素：注册时间接近、活跃时段接近、语言偏好

#### 村庄等级

| 村庄等级 | 所需贡献 | 解锁功能 |
|----------|----------|----------|
| Lv.1 新村 | 0 | 基础村内交易、串门 |
| Lv.2 小镇 | 5,000 | 村庄公告板、共建任务 |
| Lv.3 集镇 | 20,000 | 村庄集市、NPC 商人 |
| Lv.4 繁华镇 | 50,000 | 跨村集市、村庄装饰 |
| Lv.5 名望村 | 100,000 | 村庄排名榜、专属称号、村庄特产 |

#### 共建任务

- 村庄等级提升需要全村玩家共同贡献资源
- 共建项目示例：
  - **修建集市**：需要全村贡献 木材 ×500、石料 ×300、金币 ×10,000
  - **建造钟楼**：需要全村贡献 铁矿 ×200、金币 ×20,000
  - **修缮道路**：需要全村贡献 石料 ×600、金币 ×5,000
- 贡献多的玩家获得额外奖励和「建设功臣」称号

### 6.2 集市系统

#### 集市类型

| 集市 | 范围 | 开放条件 | 特色 |
|------|------|----------|------|
| 村内摊位 | 本村 | 村庄 Lv.1 | 免手续费，随时交易 |
| 村庄集市 | 本村 + NPC 商人 | 村庄 Lv.3 | NPC 出售稀有物品 |
| 跨村大集市 | 所有村庄 | 村庄 Lv.4 | 5% 手续费，物品更丰富 |

#### NPC 商人

- 每个集市日随机刷新 2-3 个 NPC 商人
- NPC 商人出售当季稀缺资源、稀有种子、装饰品
- 部分商品限量供应，先到先得

#### 稀缺资源机制

- 每个村庄有 1-2 种「村庄特产」，该类作物种植加成 +30%
- 不同村庄特产不同，促进跨村贸易需求
- 特产由村庄地理位置（随机分配）决定

### 6.3 好友系统

### 7.4 AI 架构

#### 规则引擎层（日常决策）

- 负责高频、确定性决策：种什么、何时收获、何时喂养
- 基于决策树 + 策略偏好参数
- 执行频率：每 5 分钟 tick 一次
- 零 LLM 调用成本

#### LLM 调用层（社交对话 / 复杂决策）

- 负责对话生成、关键社交事件、复杂策略建议
- 调用频率控制：每用户每天上限 10-20 次
- 支持多模型切换（根据场景复杂度选用不同模型）
- 对话缓存：相似场景复用已生成对话模板
- 降级策略：LLM 不可用时回退到预设对话模板库

#### 成本控制策略

| 策略 | 描述 |
|------|------|
| 分级调用 | 日常闲聊用轻量模型，关键事件用高能力模型 |
| 对话缓存 | 相似上下文的对话缓存复用 |
| 批量生成 | 离线期间社交行为批量异步生成 |
| 模板兜底 | 储备 200+ 对话模板作为降级方案 |
| 频率限制 | 每用户每天 LLM 调用上限 |

## 十、运营策略

### 10.1 冷启动方案

#### AI 机器人填充村庄

- 新服开启时，每个村庄预置 5-8 个 AI 机器人玩家
- AI 机器人拥有完整的 Agent 行为，与真实玩家无异
- 随着真实玩家加入，AI 机器人逐步减少（但保留 2-3 个保底）
- AI 机器人行为模式多样化，模拟不同类型玩家

#### 新手引导

- 前 3 天「教学期」：引导种植、收获、首次串门、首次交易
- 第 4-7 天「融入期」：引导加入村庄活动、结交 3 个朋友
- 第 8-14 天「成长期」：引导建筑升级、参与共建、探索集市

### 10.2 留存设计

#### 每日登录奖励

| 天数 | 奖励 |
|------|------|
| 第 1 天 | 金币 ×100 |
| 第 2 天 | 种子 ×5 |
| 第 3 天 | 体力药水 ×1 |
| 第 4 天 | 金币 ×200 |
| 第 5 天 | 友谊点 ×50 |
| 第 6 天 | 稀有种子 ×1 |
| 第 7 天 | 钻石 ×10 + 专属装饰 |

#### 赛季激励

- 赛季开始发布赛季手册（类似战令）
- 完成赛季任务获得积分和奖励
- 赛季末冲刺：最后 3 天排名分数翻倍

#### 社交留存

- Agent 每日社交日报（推送摘要到微信）
- 好友回归提醒（好友离线超过 3 天通知）
- 村庄活动提醒（共建进度、集市日）

### 10.3 社交裂变设计

#### 邀请好友

| 邀请人数 | 邀请者奖励 | 被邀请者奖励 |
|----------|-----------|-------------|
| 1 人 | 钻石 ×20 | 新手礼包 |
| 3 人 | 稀有种子 ×1 | 新手礼包 |
| 5 人 | 专属称号「社交达人」| 新手礼包 |
| 10 人 | 限定装饰 + 50 钻石 | 新手礼包 |

#### 微信分享

- 分享内容类型：
  - 农场截图（像素风明信片）
  - 丰收成就分享
  - 村庄排名分享
  - 稀有作物/动物炫耀
- 分享奖励：每日首次分享获得体力 ×20
- 好友点击分享链接可直接加入同一村庄

#### 社交裂变循环

```
玩家 A 分享丰收成就到微信
    ↓
好友 B 点击链接 → 注册并加入 A 的村庄
    ↓
A 获得邀请奖励 + B 成为 A 的邻居
    ↓
A 的 Agent 主动拜访 B → B 感受到社交乐趣
    ↓
B 也开始分享 → 裂变持续
```
